#### Original Dataset

### Custom Dataset
- Original Dataset에서 '감정_소분류', '사람문장#n', 시스템문장#n' 을 모아 새롭게 구성한 데이터 셋
- 대화는 1턴으로 끝남
- Original Dataset의 각기 다른 대화 턴의 수 때문에 새롭게 구성함
